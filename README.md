# Projekt_IoT 
Aplikacja do inteligentnego nawadniania ogrodu zintegrowana ze zbiornikiem wody deszczowej

**1. Opis Projektu**

System monitoruje poziom wody w zbiorniku na deszczówkę oraz wilgotność gleby za pomocą symulowanych czujników. Na podstawie tych danych automatycznie uruchamia podlewanie, jeśli gleba jest sucha, a w zbiorniku znajduje się wystarczająca ilość wody. Użytkownik ma podgląd na wilgotność gleby oraz na poziom wody w zbiorniku. 

**2. User Stories**

Jako właściciel ogrodu, chciałbym, aby system automatycznie podlewał rośliny, gdy gleba jest sucha, a w zbiorniku znajduje się wystarczająca ilość wody, aby nie musieć tego robić ręcznie.

Jako ogrodnik, chciałbym mieć możliwość ręcznego napełnienia zbiornika wodą z kanalizacji, gdy poziom wody w zbiorniku jest niski i nie ma opadów deszczu, aby zapewnić ciągłość podlewania.

Jako osoba dbająca o ogród, chciałbym mieć możliwość stałego podglądu stanu wilgotności gleby i poziomie wody w zbiorniku, aby móc odpowiednio reagować na potrzeby moich roślin.
